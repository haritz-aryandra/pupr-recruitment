Selamat datang di Sistem Pengadaan CPNS Kementerian PUPR Tahun 2017

Untuk bisa masuk ke tahap pendaftaran, Pelamar harus memiliki UserID dan Password (Akun Pelamar) di dalam sistem. UserID dan Password ini bisa didapatkan dengan mendaftar di website SSCN BKN (https://sscn.bkn.go.id).

Perlu diperhatikan bahwa sinkronisasi data UserID dan Password dari server SSCN BKN ke Sistem Pengadaan CPNS KemenPUPR 2017 tidak berlangsung seketika / real-time. Untuk mentransfer data Pelamar terbaru dari server SSCN BKN membutuhkan waktu minimal 10 menit, dan maksimal 1 hari kerja (1 x 24 jam).

Jadi setelah Pelamar melakukan pendaftaran biodata di website SSCN BKN, mohon bersabar untuk menunggu minimal 10 menit untuk bisa melakukan pendaftaran di Sistem KemenPUPR. Hal ini juga perlu disiasati agar tidak menunda pendaftaran, yang bisa menyebabkan terlambat mendaftar.
